## 🌍 Descrição completa do projeto

O site **Agrinho 2026** é um projeto educativo desenvolvido com o objetivo de explicar o tema:

> **“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”**

Ele apresenta informações sobre agricultura, sustentabilidade, tecnologia no campo e a importância do agronegócio para a sociedade.

O site foi estruturado em páginas e seções organizadas, permitindo uma navegação simples e intuitiva.

---

## 🧱 Como o projeto foi construído

### 📄 HTML (estrutura)

Responsável por organizar o conteúdo:

* Título principal “Agrinho 2026”
* Menu de navegação
* Seções informativas
* Rodapé

---

### 🎨 CSS (estilo visual)

Responsável pela aparência:

* Cores verdes relacionadas ao meio ambiente 🌱
* Layout moderno e centralizado
* Animações suaves
* Responsividade para celular 📱
* Efeitos de hover e transições

---

### ⚡ JavaScript (interatividade)

Responsável pelas funções dinâmicas:

* Mensagem de boas-vindas ao abrir o site
* Rolagem suave no menu
* Efeito de digitação no título
* Animações ao rolar a página
* Mensagens no console

---

## ⚙️ Como o site funciona

1. O usuário acessa a página
2. O HTML exibe todo o conteúdo
3. O CSS estiliza e organiza o visual
4. O JavaScript adiciona interações e animações
5. O usuário navega entre as seções de forma fluida
